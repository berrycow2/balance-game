# ⚖️ Симулятор весов (Balance Scale Simulator)

Интерактивный научно-образовательный симулятор классических рычажных весов с реалистичной физикой затухающих колебаний, адаптивной графикой и двумя режимами отображения грузов.

## 🎯 Назначение
- Визуализация принципа рычага и равновесия масс
- Образовательный инструмент для понимания соотношений веса
- Развлекательный тренажер логики и баланса
- Демонстрация адаптивной 2D-графики и физики в браузере

## ✨ Возможности
- 🖱️ Drag & Drop грузов на левую/правую чашу (полноэкранная зона приёма)
- 🌊 Физика маятника: плавные затухающие колебания, пропорциональный наклон
- 📐 Адаптивное масштабирование стопки: до 1000+ объектов на чаше без выхода за границы
- 🔀 Режим `Metals` (металлические гирьки) и `Emoji` (объекты природы/быта)
- 📏 Интерактивная подстройка веса: кнопки ▲▼ (+/- 1 кг, диапазон 1–50 кг)
- 🎲 Кнопка `Mix` для случайного порядка грузов в нижней панели
- 📜 Отчет в стиле старинного свитка с детализацией нагрузок
- 🔄 Мгновенный сброс состояния
- 📱 Полная поддержка мобильных устройств (touch, DPR, адаптивная верстка)
- 🎨 Визуальные эффекты: частицы при касании, свечение баланса, плавные анимации появления

## 🛠 Технологии
- `HTML5 Canvas` (2D Context)
- `Vanilla JavaScript` (ES6+)
- `CSS3` (Flex, Radial/Linear Gradients, Transitions)
- Zero dependencies, single-file architecture

## 🚀 Деплой
1. Сохраните код как `index.html`
2. Загрузите в корень директории `/balance` на сервере
3. Убедитесь, что сервер отдает файл с `Content-Type: text/html; charset=utf-8`
4. Откройте `https://mathwish.ru/balance`

## 📱 Мобильная адаптация
- Отключен нативный зум и скролл (`touch-action: none`)
- Hit-areas расширены до 24px для комфортного касания пальцем
- Canvas рендерится с учетом `devicePixelRatio`
- Адаптивный layout пересчитывается при изменении ориентации экрана
