# 🤖 PROMPT для генерации проекта "Симулятор весов"

Создай single-file HTML5 игру-симулятор классических рычажных весов. Без внешних зависимостей. Всё в одном файле `index.html`.

## 🎯 Core Requirements
1. **Физика**: Затухающий маятник. Угловая скорость зависит от разницы масс. Плавное выравнивание с колебаниями. `SPRING_K=0.025`, `DAMPING=0.96`, `MAX_ANGLE=30°`.
2. **Взаимодействие**: Drag & Drop мышью и тачем. Полноэкранная зона приёма: левая половина → левая чаша, правая → правая.
3. **Грузы**: 
   - Режим 1: 16 металлов (Медь, Золото, Платина и т.д.) с уникальными RGB-градиентами.
   - Режим 2: ~85 эмодзи (животные, фрукты, мебель) с подписями веса.
   - 17 номиналов: 1,2,3,5,7,10,12,15,17,22,27,31,32,39,45,50,75.
   - При старте случайно выбирается 17 объектов из пула.
4. **Стек & Анимации**: Грузы кладутся стопкой с случайным смещением. При снятии стопка плавно "оседает". Адаптивное масштабирование: до 5 шт = 100%, дальше плавно уменьшается до min 12%.
5. **UI**: 
   - Кнопка `⟳` сброс (активна когда чаши пусты)
   - Кнопка `😐/😄` переключение режимов
   - Кнопка `🔀 Mix` перемешивание порядка внизу
   - Кнопка `📜 Отчет` модальное окно-свиток с перечислением грузов и итогами
   - Тултип при наведении на объект внизу
6. **Интерактивная подстройка**: В режиме металлов над/под грузиком ▲▼ треугольники цвета металла. Клик ▲ +1 кг (макс 50), ▼ -1 кг (мин 1). Размер меняется динамически. В режиме эмодзи скрыты.
7. **Визуал**: Светлый бежевый фон, медная стойка, цепи с провисанием, частицы при касании, свечение зоны баланса, плавная анимация появления копий (3 сек, elastic scale).
8. **Мобильная поддержка**: `touch-action: none`, `preventDefault`, `devicePixelRatio` учет, hit-areas ≥24px, адаптивный layout при resize.

## 📐 Technical Constraints
- Только Vanilla JS + Canvas 2D. Без библиотек.
- Один `<canvas>`, один `<style>`, один `<script>`.
- `requestAnimationFrame` loop.
- Все координаты грузов на весах относительные к центру чаши (`rx`, `ry`).
- Код должен быть production-ready, комментирован, без `eval`/`document.write`.

Сгенерируй полный рабочий код.
